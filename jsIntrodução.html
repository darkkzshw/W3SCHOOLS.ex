O que o JavaScript faz?

JavaScript pode esconder elementos HTML.

JavaScript pode mudar um conteúdo HTML.

JavaScript pode mudar o estilo de um elemento HTML (nesse caso, aumentar a fonte).

JavaScript pode mostrar elementos HTML que estão escondidos.

Olá JavaScript!

JavaScript pode modificar valores atribuídos ao HTML.

Nesse caso, o JavaScript muda o valor do atributo src(source/fonte) de uma imagem.
